# Personal Site UI Kit

Interactive prototype of Manoj Achari's personal brand site.

## Screens
1. **Home** — Hero, about strip, selected work
2. **Work** — Project grid with filter tags
3. **Contact** — Minimal contact form

## Components
- `Nav` — Fixed top nav with logo + links
- `Hero` — Full-height headline section with CTA
- `AboutStrip` — Compact identity row
- `ProjectCard` — Work thumbnail card
- `ContactForm` — Email + message form
- `Footer` — Minimal footer

## Design Width
1280px desktop

## Icons
Phosphor Icons Bold weight via CDN.

## Fonts
Space Grotesk (display) + DM Sans (body) via Google Fonts.
